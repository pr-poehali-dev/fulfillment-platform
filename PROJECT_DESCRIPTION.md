# FulfillHub — Описание проекта

## Что это

**FulfillHub** — маркетплейс фулфилмент-услуг для продавцов на Wildberries, Ozon, Яндекс Маркет и других торговых площадках. Сервис помогает продавцам находить, сравнивать и выбирать подходящих партнёров по хранению и обработке товаров, а фулфилмент-центрам — получать целевые заявки от клиентов.

**Аудитория:** Продавцы на маркетплейсах, которым нужен аутсорсинг логистики и склада (малый и средний бизнес). Фулфилмент-центры, которые ищут клиентов.

**Бизнес-модель:** Лидогенерация. Покупатель бесплатно отправляет заявки фулфилментам. Фулфилменты платят за размещение и/или за лиды.

## Разделы сайта

### 1. Главная страница — Каталог (`/`)

Основной раздел сервиса. Включает:

**Hero-блок**
- Заголовок и подзаголовок с описанием сервиса
- Две точки входа:
  - «Подобрать фулфилмент за 3 минуты» — запускает квиз-опросник
  - «Добавить фулфилмент» — ведёт на страницу регистрации партнёра

**Каталог с фильтрами**
- Карточки фулфилментов с логотипом, рейтингом, ценами, набором услуг
- Умная сортировка (по рейтингу, рангу, цене, количеству отзывов, алфавиту)
- Инфинити-скролл (12 карточек за раз)
- Детальная карточка партнёра по клику (галерея, полное описание, цены, услуги)

**Система фильтрации**
- Маркетплейсы: WB, Ozon, Яндекс Маркет, СберМегаМаркет, Ali
- Схемы работы: FBS, FBO, DBS, Кросс-докинг
- Услуги (13): фотосъёмка, маркировка ЧЗ, температурный режим, работа с возвратами, опасные грузы, видеонаблюдение и др.
- Виды упаковки (7): полиэтилен, коробки, ВПП, термоусадка, деревянная обрешётка и др.
- Специализации (7): одежда/обувь, косметика, электроника, стройматериалы и др.
- Ценовые диапазоны: хранение, сборка, доставка (слайдеры от/до)
- Площадь склада (от/до)
- Минимальный рейтинг (слайдер 0–5)
- Сертификаты (мультивыбор)
- Города (мультивыбор с автодополнением)

**Навигационная панель**
- Выбор города с автоопределением по IP
- Кнопка избранного с счётчиком
- Кнопка сравнения с счётчиком
- Вход / личный кабинет

### 2. Квиз-опросник для продавцов

6-шаговый опрос помогает подобрать фулфилмент без ручной фильтрации:

| Шаг | Вопрос | Тип |
|-----|--------|-----|
| 1 | Категория товара (25 вариантов) | Мультивыбор |
| 2 | Формат работы (FBO / FBS / DBS / B2B / B2C) | Мультивыбор |
| 3 | Ежемесячный объём (от 100 до 100 000+ ед.) | Одиночный |
| 4 | Дополнительные услуги | Мультивыбор |
| 5 | Виды упаковки | Мультивыбор |
| 6 | Контактные данные (имя, телефон, email) | Форма |

После заполнения — данные уходят в систему, менеджер подбирает партнёров.

---

### 3. Форма запроса коммерческого предложения

Пользователь отправляет одну форму сразу нескольким фулфилментам. Поля:
- Имя (обязательно)
- Компания (опционально)
- Email (обязательно)
- Телефон (обязательно)
- Количество SKU (опционально)
- Количество заказов в месяц (опционально)
- Комментарий (опционально)
- Согласие на обработку персональных данных и условия сервиса

После отправки — экран подтверждения со списком партнёров-получателей и предложением зарегистрироваться для отслеживания заявок.

---

### 4. Сравнение партнёров

Пользователь добавляет до 3 партнёров и видит их параллельно по:
- Рейтинг и отзывы
- Год основания, площадь склада, штат
- Маркетплейсы, схемы работы, упаковка
- Цены на хранение, сборку, доставку
- Услуги и специализации
- Кнопка «Запросить КП» прямо из сравнения

---

### 5. Избранное

- Сохраняется в localStorage (без регистрации)
- Панель со списком избранных партнёров
- Фильтр «Показать только избранных» в каталоге
- Кнопка «Запросить КП у всех избранных»

---

### 6. Личный кабинет продавца (`/seller`)

Раздел для продавцов, которые отправили заявки:
- История заявок (КП) с датами и статусами
- Список фулфилментов, которым отправлены заявки
- Статус обработки каждой заявки
- Ответы и коммуникация с партнёрами

---

### 7. Кабинет фулфилмент-центра (`/admin`)

Панель для партнёров-фулфилментов:
- Управление профилем (название, описание, фото, контакты)
- Настройка услуг, цен, площади склада
- Управление несколькими точками/складами
- Входящие заявки (КП) от продавцов с контактами
- Статистика просмотров и заявок
- Загрузка фото, логотипа
- Управление дополнительными услугами с ценами

---

### 8. Регистрация фулфилмента (`/for-fulfillment`)

Страница для фулфилмент-центров, которые хотят разместиться:
- Описание преимуществ размещения
- Форма заявки на добавление в каталог
- Информация о тарифах и условиях

---

### 9. Страница модерации (`/moderation`)

Внутренний инструмент:
- Очередь новых заявок на регистрацию фулфилментов
- Просмотр данных и проверка
- Подтверждение / отклонение

---

### 10. Калькулятор фулфилмента (`/kalkulator-fulfillmenta`)

Инструмент расчёта стоимости:
- Количество товаров и SKU
- Объём заказов в месяц
- Расчёт приблизительных затрат на хранение, сборку и доставку
- SEO-страница с полезным контентом

---

### 11. SEO-страницы

Статические контентные страницы для органического трафика:
- `/fulfillment/moskva` — фулфилмент в Москве (с отфильтрованным каталогом)
- `/fulfillment/sankt-peterburg` — фулфилмент в Санкт-Петербурге
- `/fulfillment/wildberries` — фулфилмент для Wildberries
- `/fulfillment/ozon` — фулфилмент для Ozon
- `/chto-takoe-fulfillment` — что такое фулфилмент (образовательная)
- `/fbo-vs-fbs` — сравнение схем FBO и FBS (образовательная)

---

### 12. Партнёрская программа (`/sales`)

Страница для привлечения агентов/партнёров:
- Условия сотрудничества
- Форма заявки на партнёрство

## Техническая архитектура

| Слой | Технологии |
|------|------------|
| Фронтенд | React 18, TypeScript, Vite, Tailwind CSS, shadcn/ui |
| Бэкенд | Python 3.11, Cloud Functions (бессерверные) |
| База данных | PostgreSQL |
| Файловое хранилище | S3-совместимое (CDN: cdn.poehali.dev) |
| Аутентификация | Кастомная (токены, cookie-сессии) |
| Аналитика | Яндекс.Метрика (цели: квиз, КП, регистрация) |
| Геолокация | Автоопределение города по IP через отдельную Cloud Function |

**Бэкенд-функции:**
- `fulfillment` — CRUD фулфилментов, модерация, фото, КП, email-уведомления
- `auth` — регистрация, вход, управление сессиями
- `detect-city` — определение города по IP
- `subscribe-email` — подписка на рассылку
- `get-subscribers` — список подписчиков для рассылок
- `fetch-og-images` — автоматическая загрузка OG-картинок с сайтов партнёров

## Текущее состояние базы данных

**Основная таблица `fulfillments`:**
- Одобренных партнёров: ~70+
- Поля: название, описание (2 блока), город, год основания, площадь склада, штат, режим работы, сайт, телефон, email
- Цены: хранение, сборка, доставка (строковый + числовой форматы), минимальный объём
- Массивы: фото, маркетплейсы, услуги, схемы работы, упаковка, сертификаты, специализации
- Дополнительные услуги: JSON-массив с названием, описанием и ценой
- og_image: автоматически загружается с сайта партнёра
- Рейтинг, количество отзывов, ранг для сортировки

**Таблица `quotes`:** заявки от продавцов к фулфилментам со статусами

**Таблица `users`:** аккаунты продавцов и фулфилмент-центров с ролями

## Варианты улучшений и доработок

### 🟢 Быстрые (1–3 дня)

**UX / Конверсия**
- Быстрые фильтр-теги под навбаром (FBS, Фото, Маркировка ЧЗ) — один клик без открытия панели
- Упрощение формы КП — только имя + телефон + email в основной форме, остальное в «развернуть»
- Кнопка «Отправить заявку всем» прямо в шапке каталога
- Показывать WhatsApp/Telegram кнопку на карточке, если у партнёра есть контакт

**Данные**
- Автозапуск `fetch-og-images` по расписанию (CRON раз в неделю)
- Счётчик «сколько продавцов запросили КП» на карточке (социальное доказательство)
- Заполнить пустые цены у партнёров — сейчас у многих «по запросу»

---

### 🟡 Средние (1–2 недели)

**Квиз → Каталог**
- После квиза автоматически открывать каталог с применёнными фильтрами вместо «менеджер свяжется»
- Показывать «Подобрано X партнёров под ваш запрос» после квиза

**Онбординг**
- 2–3 вопроса при первом визите (маркетплейс + город + схема) → сразу фильтруют каталог без регистрации
- Подсказки-тултипы рядом с терминами FBO/FBS/DBS для новичков

**Кабинет фулфилмента**
- Email-уведомления при новой заявке (сейчас ли работает — проверить)
- Статистика в кабинете: сколько просмотров, кликов, заявок за период

**Продавец**
- Статусы заявок с таймлайном: «Отправлено → Просмотрено → Ответили»
- Push/email уведомление при ответе фулфилмента

---

### 🔵 Стратегические (1–2 месяца)

**Отзывы и репутация**
- Возможность для продавцов оставлять отзывы после работы с фулфилментом
- Верификация отзывов (только от реальных клиентов)
- Раздел отзывов на детальной карточке

**Тарифы и монетизация**
- Платное поднятие в каталоге (featured-размещение)
- Тарифы для фулфилментов: бесплатный (базовое размещение) / платный (приоритет, расширенный профиль)
- Аналитика для фулфилментов (показы, CTR, конверсия в заявки) как платная функция

**Экосистема**
- Сравнение как отдельная страница (не модалка) — удобнее делиться ссылкой
- Блог / гайды по фулфилменту (SEO + экспертиза)
- Интеграция с маркетплейсами — автоматический расчёт затрат по API WB/Ozon
- Telegram-бот для уведомлений и быстрых ответов на заявки

**Масштабирование**
- API для агрегаторов
- Мобильное приложение (PWA)
- Карта складов — визуализация фулфилментов на карте России
- Мультиязычность (для зарубежных продавцов)

## Ключевые показатели для отслеживания

| Метрика | Описание |
|---------|---------|
| Конверсия в заявку | % посетителей → отправили хотя бы 1 КП |
| Квиз-завершение | % начавших квиз → дошли до конца |
| Заявок на фулфилмент | Сколько КП в среднем получает 1 партнёр в месяц |
| Time-to-first-quote | Время от первого визита до отправки заявки |
| Возврат продавцов | % вернувшихся в кабинет после регистрации |
| Активация фулфилментов | % зарегистрировавшихся → получили первую заявку |
